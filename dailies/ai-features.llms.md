# AI features

Overview of the AI features available in Positron.

AI features in Positron help you write, understand, and debug code. To start using them, follow the [Get started](assistant-getting-started.llms.md) guide.

### Posit Assistant

Chat about your code, Console sessions, and in-memory objects.

### Completions

Inline code suggestions as you type, powered by Posit AI NES or GitHub Copilot.

### Notebook AI features

Notebook-aware AI that understands your cells, outputs, and execution history.

### Console fix and explain

Fix and explain errors from your R and Python Console sessions.

### Quarto fix and explain

Fix and explain errors from Quarto chunks running with inline output.

### Git commit messages

Generate a commit message for your staged changes from the Source Control pane.
